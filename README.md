# 🚀 Portfolio Website

A modern and responsive personal portfolio website built with **React** to showcase my projects, skills, and previous work in an elegant and interactive way.

## ✨ Features

* 📂 Display my personal and professional projects.
* 🎨 Clean and responsive user interface.
* 🖼️ Interactive project sliders using Swiper.
* 🔄 Smooth navigation between pages with React Router.
* 🎯 Modern icons powered by React Icons.
* 📱 Fully responsive design for desktop, tablet, and mobile devices.

## 🛠️ Technologies Used

* React
* Tailwind CSS
* React Router DOM
* Swiper.js
* React Icons

## 📸 About the Project

This portfolio was created to present my previous work and projects in a professional format. It focuses on performance, responsiveness, and user experience while providing visitors with an easy way to explore my projects and skills.

## 🚀 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/your-repository.git

cd your-repository

```


# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled.
